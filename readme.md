# 📊 成績計算系統

一個功能完整的網頁成績計算工具，支援 Excel 匯入匯出、自訂權重比例、自動選取最高分等功能。

## ✨ 功能特色

- 📁 **Excel 匯入**：支援 .xlsx 格式匯入學生成績
- 🏆 **智能選取**：自動為每位學生選取最高分的 N 次平時成績
- ⚖️ **自訂權重**：自由設定平時成績與作業成績的佔比（0%～100%）
- 📊 **即時計算**：一鍵結算總成績，顯示完整計算過程
- 💾 **Excel 匯出**：匯出只包含被選中的最高分成績，保留原始作業成績
- 🎨 **直覺介面**：美觀的漸層配色，支援 RWD 響應式設計
- 📱 **跨平台**：純網頁技術，手機、平板、電腦均可使用

## 🚀 立即使用

🔗 **線上版**：https://你的帳號.github.io/grade-calculator

## 📋 匯入檔案格式

### Excel 檔案欄位規則

| 欄位順序 | 欄位名稱 | 說明 | 必填 |
|---------|---------|------|-----|
| 第1欄 | 班級 | 學生的班級編號 | ✅ |
| 第2欄 | 座號 | 學生的座號 | ✅ |
| 第3欄 | 姓名 | 學生姓名 | ✅ |
| 第4欄起 | 平時成績 | 多次平時考試成績 | ✅ (至少1次) |
| 倒數第3欄 | 作業成績1 | 第一次作業分數 | ✅ |
| 倒數第2欄 | 作業成績2 | 第二次作業分數 | ✅ |
| 倒數第1欄 | 作業成績3 | 第三次作業分數 | ✅ |

> ⚠️ **重要規則**：
> - 第一列必須是標題列（欄位名稱）
> - 平時成績可以有任意欄數（至少1欄）
> - 最後3欄固定為作業成績
> - 空白儲存格自動視為 0 分
